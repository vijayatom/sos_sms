
## SOS SMS
## Installation
#### Create Environment 
    python -m venv /path/to/new/virtual/environment
#### Activate Environment
#### for windows
    enviroment/include/activate      
    
#### for linux
    source enviroment/bin/activate      
    
#### Install essential libraries and packages:
    pip install -r requirements.txt
